favorites_app
=============

## ABOUT

A funny, simple webapp for saving your favorite Matt Damon movies. 

## Installation / Setup

Clone repo locally. Install gems:

    bundle install

Set up database. We are using sqlite3.

    rake db:create db:migrate

Run your local server.

    shotgun
